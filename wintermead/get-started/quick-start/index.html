Basic Template

You can set up the Wintermead Web Framework and be on your way in under a minute. Just add this code to your web page for the basic template to take effect immediately. 

<!doctype html>
<html>
  <head>
    <meta charset='utf-8'>
    <meta name='viewport' content='width=device-width, initial-scale=1'>
    <meta http-equiv='X-UA-Compatible' content='ie=edge'>

    <title>Basic Template</title>

    <!-- Wintermead CSS -->
    <link rel="stylesheet" href="project-folder/assets/styles/wintermead.min.css">
  </head>

  <body>
    <h1>Hello, World!</h1>
  </body>
</html>
